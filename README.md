# How_Fast_You_Can_Count_2
🚀 How Fast Can You Count?
How Fast Can You Count adalah game web Mental Math interaktif yang menantang kecepatan berhitung dan memori jangka pendek pemain. Berbeda dengan kuis matematika biasa, di sini angka akan muncul secara mengalir (streaming) dan pemain harus menjaga total hitungannya di luar kepala hingga akhir sesi.

✨ Fitur Utama
Mode Permainan:
Solo Mode: Latih kecepatanmu sendiri dan pecahkan rekor pribadimu.
Multiplayer (Mabar): Duel secara real-time dengan teman menggunakan sistem Room ID (Powered by Firebase).
Mekanik Aliran Angka (Random Stream): Angka muncul satu per satu dengan operator acak di depannya. Fokus adalah kunci!
Customizable Difficulty: * Level: Easy, Medium, Hard (Mengatur besaran angka dan kecepatan aliran).
Operator: Pilih +, -, *, atau / sesuai keinginanmu.
Sistem Penilaian Presisi: Skor tidak hanya berdasarkan benar/salah, tapi seberapa dekat jawabanmu dengan hasil asli (Fuzzy Logic Scoring).
Real-time Leaderboard: Lihat siapa yang memiliki kalkulator tercepat di otaknya secara global.

🛠️ Tech Stack
Frontend: HTML5, CSS3 (Cyberpunk Glassmorphism UI), JavaScript (ES6+).
Backend/Database: Firebase Realtime Database untuk sinkronisasi mabar yang instan.

🎮 Cara Bermain
Buka web game.
Pilih Mode (Solo atau Mabar).
Pilih tingkat kesulitan dan operator matematika yang ingin diuji.
Perhatikan layar! Angka akan muncul bergantian (Contoh: 5 -> +10 -> -3...).
Hitung totalnya di kepala tanpa alat bantu.
Input hasil akhirmu secepat mungkin.
Lihat skormu berdasarkan akurasi dan kecepatan!
